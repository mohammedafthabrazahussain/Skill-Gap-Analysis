# SkillGap.AI

### Smart Skill Gap Analysis & Career Roadmap Platform

SkillGap.AI is a modern web application that helps users analyze their current skills, identify gaps for a target career role, and follow a structured learning roadmap to achieve their goals.

The platform provides a clear, data-driven approach to career development through an interactive and user-friendly interface.

---

## What This Project Does

Many learners struggle with questions such as:

* What skills are required for a specific job role?
* How prepared am I for a career?
* What should I learn next?

SkillGap.AI addresses these challenges by:

* Comparing user skills with industry requirements
* Calculating readiness level
* Providing a structured roadmap for improvement

---

## How It Works

### 1. Select Your Goal

Choose a target role (for example, Frontend Developer).

### 2. Mark Your Skills

Select the skills you already know using an interactive checklist.

### 3. Get Instant Analysis

The system:

* Calculates your Readiness Score
* Identifies missing skills
* Highlights priority areas

### 4. Follow the Roadmap

Receive:

* Step-by-step learning guidance
* Recommended resources
* Estimated time to achieve the goal

## Getting Started

### 1. Clone the Repository

```bash
git clone <your-repo-link>
cd SkillGap.AI
```

---

### 2. Backend Setup

```bash
cd server
npm install
node seed_pro.js
node index.js
```

---

### 3. Frontend Setup

```bash
cd client
npm install
npm run dev
```

---

### 4. Open the Application

http://localhost:5173
